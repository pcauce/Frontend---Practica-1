JavaScript

Introduce interactividad y dinamismo en una página web. A diferencia de HTML y CSS, que son lenguajes declarativos, JavaScript es un lenguaje interpretado que se ejecuta a través del motor de JavaScript del navegador y eventos:

  • El motor de JavaScript: cada navegador emplea su propio motor de JavaScript (como V8 en Chrome o SpiderMonkey en Firefox). Este motor ejecuta el código JavaScript mediante interpretación Just-In-Time (JIT). En este proceso, el código es convertido a bytecode o código máquina en tiempo real, optimizando así su ejecución.
  • Event Loop: hace referencia a el uso que hace el navegador de un modelo basado en eventos que permite la gestión de la ejecución de JavaScript, de modo que, es un mecanismo que permite gestionar operaciones asíncronas, tales como eventos de usuario, temporizadores y solicitudes de red, dando garantía de una ejecución no bloqueante del código JavaScript.
  • Manipulación del DOM: JavaScript ofrece la posibilidad de manipular el DOM en tiempo real, lo que permite la realización de cambios dinámicos en la estructura y contenido de la página sin necesidad de volver a tener que cargarla de nuevo, redundando en una mejora importante en la experiencia del usuario.

Principales Características

  • Interpretado y ejecutado por el navegador.
  • Manipulación del DOM en tiempo real.
  • Compatibilidad con APIs.

Ventajas

  • Aporta interactividad con el usuario.
  • Soporta varios paradigmas de programacion.
  • Comunidad muy amplia.

Frameworks y librerías

React

Librería desarrollada por Facebook, diseñada para construir interfaces de usuario, especialmente en aplicaciones de una sola página. Permite crear componentes reutilizables que gestionan su propio estado.

Angular

Framework desarrollado por Google, que permite crear aplicaciones web dinámicas y robustas. A diferencia de React, Angular es un framework completo que ofrece una solución integral para el desarrollo de aplicaciones.

Vue

Framework que se enfoca en la construcción de interfaces de usuario. Es conocido por su simplicidad y flexibilidad, lo que lo convierte en una opción popular tanto para proyectos pequeños como para aplicaciones grandes y complejas.
