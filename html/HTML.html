HTML (HyperText Markup Language)

Es un lenguaje de marcado que establece la estructura y el contenido de una página web. Cuando el navegador recibe el archivo, se inician una serie de procesos:

  • Parsing: el navegador descompone el HTML de forma estructurada llamada de árbol, que recibe el nombre de DOM (Document Object Model), de modo que cada etiqueta HTML constituye un nodo dentro de la estructura. La utilidad del DOM es que representa de modo interno la página web, que es utilizada por el navegador para renderizar el contenido en pantalla.
  • Renderización: el navegador utiliza el DOM para renderizar la página en la interfaz del usuario, proceso que asegura que el contenido se muestre de forma estructurada y coherente.

Principales Características

  • Organiza el contenido en elementos anidados, como encabezados, párrafos y listas, proporcionando la estructura del documento.
  • Utiliza etiquetas semánticas para describir el propósito del contenido.
  • Es compatible con todos los navegadores modernos.

Ventajas

  • Simplicidad
  • Estandarización
  • Etiquetas Semánticas
