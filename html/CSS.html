CSS (Cascading Style Sheets)

Es un lenguaje Lenguaje que define el estilo y la presentación del contenido HTML en una página o sitio web, el navegador recibe las hojas de estilo y realiza una serie de acciones o pasos como:

  • Aplicación de estilos: el navegador utiliza el DOM combinado con las reglas de estilo definidas en el CSS para formar un Render Tree, definiendo cómo debe presentarse cada elemento.
  • Cálculo del Layout: cuando el Render Tree está completo, el navegador realiza el cálculo para el layout, a lo largo del proceso determina las posiciones exactas y dimensiones de cada elemento en la pantalla.
  • Pintado: finalmente, el navegador pinta los elementos en la pantalla según las propiedades de estilo calculadas. Este paso convierte los cálculos en la visualización final que ve el usuario.

Principales Características

  • Permite aplicar estilos a elementos específicos de HTML utilizando selectores.
  • Ofrece la posibilidad de crear diseños responsive, adaptables y receptivos que se ajustan a diferentes tamaños de pantalla
  • Los estilos pueden utilizarse en cascada, de modo que permiten heredarse y sobrescribirse, lo que ofrece la posibilidad de tener un control preciso sobre la apariencia de la página.

Ventajas

  • Separación entre el contenido y sus estilos.
  • Reutilización de estilos.
  • Flexibilidad

Preprocesadores y Postprocesadores

Sass

Preprocesador de CSS que extiende las capacidades de CSS, permitiendo el uso de variables, anidación de reglas, mixins, y más.

PostCSS

A diferencia de los preprocesadores, PostCSS se aplica después de que el CSS ha sido escrito, proporcionando una mayor flexibilidad y control sobre la forma en que se procesan los estilos.
